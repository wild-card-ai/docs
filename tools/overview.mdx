Wildcard provides a comprehensive tool system that allows AI agents to interact with various APIs. Each tool represents an API endpoint with specific functionality, making it easy for agents to perform actions across different services.

## Tool System

The tool system consists of three main components:

1. **Action Class**: Type-safe enum-based system for referencing API endpoints
2. **Tool Schemas**: OpenAPI-based schemas defining tool parameters and responses
3. **API Services**: Collection of supported APIs and their authentication methods
